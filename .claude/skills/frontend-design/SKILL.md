This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic AI aesthetics.

## Design Thinking
Before coding, commit to a BOLD aesthetic direction: brutally minimal, maximalist, retro-futuristic, luxury/refined, playful, editorial, brutalist, art deco, etc.

## Frontend Aesthetics Guidelines
- Typography: Distinctive fonts, never Arial/Inter/Roboto
- Color: Cohesive theme with CSS variables, dominant colors with sharp accents
- Motion: CSS animations, scroll-triggering, hover states
- Spatial Composition: Asymmetry, overlap, diagonal flow, grid-breaking elements
- Backgrounds: Gradient meshes, noise textures, geometric patterns, dramatic shadows

NEVER use generic AI aesthetics (purple gradients, system fonts, predictable layouts).
